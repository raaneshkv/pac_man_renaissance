# Pac-Renaissance

A modern twist on the classic Pac-Man game built with Python and Pygame.

## Features
- Classic maze gameplay with pellets and ghosts
- Power-ups giving temporary invincibility
- Player flashes green while invincible
- Distinct colored ghosts
- Story intro with username input
- Game Over and restart functionality

## Requirements
- Python 3.x
- pygame 2.1+
